# 技小助云端版 - 快速开始

## 🚀 开始部署（3选1）

### ⭐ 方案 1：Windows - 简化脚本（推荐）

**双击 `deploy-simple.bat`**

---

### 方案 2：任何系统 - Vercel 网站（最简单）

1. 访问 https://vercel.com
2. 登录（推荐 GitHub）
3. New Project → Upload
4. 拖拽整个文件夹
5. Deploy

---

### 方案 3：手动命令行

```cmd
cd /d "你的路径\cloud-deployment"
npm install -g vercel
vercel login
vercel --prod
```

---

## ⚠️ 遇到乱码？

如果 `deploy.bat` 显示乱码：
- 使用 `deploy-simple.bat`（推荐）
- 或查看 `QUICK_FIX.md`
